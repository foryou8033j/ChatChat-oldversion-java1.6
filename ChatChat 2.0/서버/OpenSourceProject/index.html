Documents of ChatChat

 

Chatchat 채팅 클라이언트, 서버를 개발/구축하면서 얻게 된 지식을 문서로 정리하고자 하여 작성을 시작하였다.

기존 Chatchat 버전 (0.1 ~ 3.5) 이하 OldVersion 이라 명하는 Chatchat은 Java Swing을 기반반으로 제작되었으며,

Swing 자체적인 버그로 인하여 더 이상의 소프트웨어적 최적화가 힘들다고 판단하여 jdk1.8 에서부터 기본 지원되는 JavaFX를 기반으로 NewChat기발을 시작하였다.
